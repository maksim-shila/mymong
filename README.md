# mymong

Phaser-игра на Vite.

**Сборка и запуск**
1. `npm install`
2. `npm run dev` — локальный запуск с HMR
3. `npm run build` — сборка проекта
4. `npm run preview` — локальный просмотр собранной версии

**Управление**
- `A` / `D` или `←` / `→` — движение панели влево/вправо
- `Shift` — ускорение движения панели (boost)
- `Space` — `launch` (если шарик на панели) или `push` (если шарик уже в игре)
- `P` — пауза

## Desktop build (Electron)

1. `npm install`
2. `npm run dev:app` - run desktop app in development mode
3. `npm run dist:app` - build desktop app into `release/`
